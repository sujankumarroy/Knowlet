SECTION-A: Very Short Answer Type

Question 1

[1 Mark]

What is data?

Data refers to raw, unorganized facts, figures, or symbols that need to be processed to be meaningful.

Question 2

[1 Mark]

What is process?

A process is a series of actions or steps taken on data to convert it into useful information.

Question 3

[1 Mark]

What is information?

Information is data that has been processed, organized, and structured so that it is meaningful and useful to the recipient.

Question 4

[1 Mark]

What is system?

A system is a collection of interrelated components working together toward a common goal by accepting inputs and producing outputs.

Question 5

[1 Mark]

What is Template?

A template is a pre-formatted document or file that serves as a starting point for a new document, saving time on layout and design.

Question 6

[1 Mark]

What is AutoText?

AutoText is a feature in word processors that allows users to store and quickly reuse frequently used text or graphics.

Question 7

[1 Mark]

What is footnote?

A footnote is a note placed at the bottom of a page that provides additional information or cites a source for a specific part of the text above it.

Question 8

[1 Mark]

What is reference?

A reference is a citation to a source of information (books, articles, etc.) used in a document to give credit and allow readers to locate the source.

Question 9

[1 Mark]

What is PowerPoint?

PowerPoint is a presentation software developed by Microsoft that uses slides to display information, often including text, images, and multimedia.

Question 10

[1 Mark]

What is animation?

Animation refers to visual effects applied to individual items on a slide (like text or images) to control their movement and appearance during a presentation.

Question 11

[1 Mark]

What is the function of Clip Art?

The function of Clip Art is to provide a collection of pre-made images and illustrations that users can insert into documents or presentations to enhance visual appeal.

Question 12

[1 Mark]

What is multimedia?

Multimedia is the integration of multiple forms of media, such as text, audio, video, graphics, and animation, into a single application or presentation.

Question 13

[1 Mark]

What is worksheet?

A worksheet is a single page or grid of cells within a spreadsheet file where data is entered and manipulated.

Question 14

[1 Mark]

What is Spreadsheet?

A spreadsheet is a software application (like Excel) used for organizing, analyzing, and storing data in tabular form (rows and columns).

Question 15

[1 Mark]

What is Flash fill?

Flash fill is a spreadsheet feature that automatically senses patterns in data entry and fills the remaining cells accordingly.

Question 16

[1 Mark]

What is conditional formatting?

Conditional formatting is a feature that changes the appearance of cells (like background color or font style) based on specific conditions or criteria.

Question 17

[1 Mark]

What is mathematical function in Spreadsheet?

Mathematical functions are built-in formulas in spreadsheets used to perform calculations such as addition, subtraction, and multiplication (e.g., SUM, PRODUCT).

Question 18

[1 Mark]

What is financial function in Spreadsheet?

Financial functions are built-in formulas used to perform financial calculations like interest rates, loan payments, and investment valuations (e.g., PMT, FV).

Question 19

[1 Mark]

What is statistical function in Spreadsheet?

Statistical functions are formulas used to analyze sets of data to find trends, averages, or probabilities (e.g., AVERAGE, COUNT, MEDIAN).

Question 20

[1 Mark]

What is database?

A database is an organized collection of structured information or data, typically stored electronically in a computer system for easy access and management.

SECTION-C: Descriptive Answer Type

Answer any five of the following questions. (5 x 5 = 25 Marks)

Question 31

[5 Marks]

Discuss the applications of data processing in business.

Data processing is vital in modern business for converting raw facts into actionable insights. Key applications include:

  • Inventory Management: Tracking stock levels and predicting reorder times.
  • Payroll Processing: Calculating wages, taxes, and deductions for employees.
  • Customer Relationship Management (CRM): Analyzing customer data to improve sales and service.
  • Financial Reporting: Generating balance sheets and profit/loss statements from transaction data.

Question 34

[5 Marks]

Discuss the mail merge operation with an example.

Mail merge is a feature that allows users to create multiple personalized documents (like letters) from a single template and a data source (like a list of names).

Example: Sending a "New Year" greeting to 100 clients.

  1. Create a Main Document (the letter template).
  2. Prepare a Data Source (Excel list with client names and addresses).
  3. Insert Merge Fields (e.g., <<Name>>) into the template.
  4. Execute the Merge to generate 100 unique letters.

Question 35

[5 Marks]

Discuss the process of preparing business presentation for a new product (using PowerPoint steps).

  1. Slide 1 (Title): Product name and company logo.
  2. Slide 2 (Introduction): Problem statement or market need.
  3. Slide 3 (Features): Key benefits of the new product using bullet points.
  4. Visuals: Insert high-quality images of the product.
  5. Transitions & Animations: Apply professional transitions to maintain audience engagement.

Question 37

[5 Marks]

Discuss the application of Spreadsheet.

Spreadsheets are versatile tools in business for:

  • Budgeting: Creating annual or project-based financial plans.
  • Data Analysis: Using statistical functions to find sales trends.
  • Visual Representation: Converting data into charts and graphs for presentations.
  • Automation: Using macros or complex formulas to speed up repetitive tasks.

Question 38

[5 Marks]

How may we insert charts, graphs and formula in Spreadsheet?

  • Formula: Click a cell, type "=" followed by the calculation (e.g., =A1+B1) or a function.
  • Charts and Graphs:
    1. Select the data range to be visualized.
    2. Go to the Insert tab on the ribbon.
    3. Select the desired chart type (e.g., Bar, Pie, or Line).
    4. Customize the chart with titles and labels.

Exam Success Toolkit

Exam Tips

  • For Computer Applications, always use technical terms like "Ribbon," "Cell Reference," or "Data Source."
  • In descriptive answers, list step-by-step processes (like Mail Merge) clearly.

Important Formulas List

  • SUM: =SUM(Range)
  • AVERAGE: =AVERAGE(Range)
  • IF: =IF(Condition, Value_if_true, Value_if_false)

Answer Presentation Strategy

Keep very short answers to the point (1-2 sentences). For long answers, use bullet points for features and numbered lists for procedures. Always include examples where requested.
